# JavaScript Helpers

Project live preview on http://kbanach.github.io/JavaScript-Helpers

## How to run

First of all install it with
```bash
yarn
```

Then simply:
```bash
yarn start
```

Which will open the browser.

## Deployment to gh-pages

Every merge to `master` is deployed by Travis CI.
